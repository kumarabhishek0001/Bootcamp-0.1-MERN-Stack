# What is a promise
A promise in javascript is an object that represents the eventual result of an asynchronus operation - either a success(resolved) or failure(rejected).

It is commonly used for things like:
- Fetching data from an API.
- Reading files.
- Timers(setTimeout).
- Database Requests.

## Promise States
- Pending -> the operation still running.
- Fulfilled(resolved) -> the operation completed successfully.
- Reject -> The operation.
